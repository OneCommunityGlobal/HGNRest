# HGN Rest server

REST API used for Highest Good Network App V2,

## Installation

```bash
$ npm install gulp-cli -g
$ npm i
```

### Environment setup

We need to create a `.env` file with similar to
```bash
MONGO_URI=localhost/DEV

```

## Running the server

```bash
$ gulp
```
